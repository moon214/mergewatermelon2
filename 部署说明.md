### 方式 2: SCP 传输

```bash
# 从本地电脑执行
scp -r user@server:/root/.openclaw/workspace/MergeWatermelon2 ~/Projects/
```

### 方式 3: 压缩下载

```bash
# 在服务器压缩
cd /root/.openclaw/workspace/
tar -czf MergeWatermelon2.tar.gz MergeWatermelon2/

# 下载 (从本地执行)
scp user@server:/root/.openclaw/workspace/MergeWatermelon2.tar.gz ~/Downloads/
```
