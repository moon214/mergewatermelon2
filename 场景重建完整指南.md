## 📋 详细操作步骤

### 步骤 1: 关闭 Cocos Creator

确保 Cocos Creator 完全退出（检查任务管理器）

### 步骤 2: 运行重建脚本

**Windows:**
```powershell
cd D:\all_store\gitxiangmu\mergewatermelon2
tools\重建场景.bat
```

**或手动运行 Node.js 脚本:**
```powershell
node tools\rebuild-scene.js
```

### 步骤 3: 打开 Cocos Creator

1. 启动 Cocos Creator 3.8.8
2. 打开 `mergewatermelon2` 项目
3. 等待资源导入完成（进度条走完）

### 步骤 4: 刷新脚本

```
菜单栏 → 开发者 → 刷新脚本
```

### 步骤 5: 打开场景

1. 资源管理器 → `assets/scenes/Game.scene`
2. 双击打开

### 步骤 6: 配置组件（⚠️ 重要）

场景已重建，但需要手动配置组件引用：

#### 6.1 选择 GameManager 节点

在层级管理器中点击 `Canvas/GameManager`

#### 6.2 添加组件

在属性检查器中：
```
添加组件 → TypeScript → GameManager
添加组件 → TypeScript → FruitManager
添加组件 → TypeScript → ScoreManager
添加组件 → TypeScript → AudioManager
添加组件 → TypeScript → DouyinSDK
```

#### 6.3 配置 GameManager 属性

| 属性 | 赋值 |
|------|------|
| **Fruit Prefab** | 拖入 `assets/prefabs/Fruit.prefab` |
| **Spawn Point** | 拖入 `Canvas/SpawnPoint` 节点 |
| **Fruit Container** | 拖入 `Canvas/FruitContainer` 节点 |

#### 6.4 配置 UI 组件

1. 选择 `Canvas/UI` 节点
2. 添加组件 → TypeScript → GameUI
3. 配置各个 UI 属性（Label、Button 等）

## 🎯 Fruit 预制体配置

### 打开预制体

1. 资源管理器 → `assets/prefabs/Fruit.prefab`
2. 双击打开预制体编辑

### 添加组件

```
1. 选择根节点 "Fruit"
2. 添加组件 → TypeScript → Fruit
3. 添加组件 → 渲染 → Sprite
4. 添加组件 → 物理 2D → Circle Collider 2D
5. 添加组件 → 物理 2D → RigidBody 2D
```

### 配置属性

**RigidBody 2D:**
- Type: Static (初始为静态)
- Gravity Scale: 1.0
- Linear Damping: 0.5
- Angular Damping: 0.5

**Circle Collider 2D:**
- Radius: 25 (根据水果大小调整)
- Restitution: 0.3
- Friction: 0.8

### 保存预制体

```
Ctrl+S 保存
点击返回场景按钮
```

## ⚠️ 常见问题

### Q1: 刷新脚本后仍然报错

**解决:**
```
1. 完全关闭 Cocos Creator
2. 删除 library/, temp/, local/
3. 重新打开项目
4. 再次刷新脚本
```

### Q2: 组件无法添加到节点

**解决:**
```
1. 确保脚本文件在 assets/scripts/ 目录下
2. 确保脚本有 @ccclass 装饰器
3. 刷新脚本
```

### Q3: 预制体打开后是空的

**解决:**
```
1. 删除 assets/prefabs/Fruit.prefab
2. 重新运行重建脚本
3. 在 Cocos 中重新添加组件
```

### Q4: 水果不掉落

**解决:**
```
1. 检查 Fruit 预制体的 RigidBody 2D
2. 确保 release() 方法被调用
3. 检查控制台是否有错误
```

### Q5: 纹理加载失败

**解决:**
```
1. 确保水果 PNG 文件在 assets/textures/fruits/
2. 检查文件名是否正确
3. 等待资源导入完成
```
