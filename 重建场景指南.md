# 重建场景指南 - 合成大西瓜 2.0

**问题:** 旧场景文件版本不兼容，需要重新创建

---

## 🔧 方案 A: 清理缓存（先试这个）

### 步骤 1: 关闭 Cocos Creator

确保 Cocos Creator 完全关闭

### 步骤 2: 删除缓存目录

```powershell
cd D:\all_store\gitxiangmu\mergewatermelon2

# 删除缓存
Remove-Item -Recurse -Force library\
Remove-Item -Recurse -Force temp\
Remove-Item -Recurse -Force local\
```

### 步骤 3: 重新打开项目

1. 打开 Cocos Creator 3.8.8
2. 打开项目 `mergewatermelon2`
3. 等待资源导入完成（看进度条）
4. 菜单 → 开发者 → 刷新脚本

### 步骤 4: 检查场景

1. 双击 `assets/scenes/Game.scene`
2. 如果场景能打开 → ✅ 成功
3. 如果仍有错误 → 执行方案 B

---

## 🏗️ 方案 B: 手动重建场景

### 步骤 1: 创建新场景

1. 菜单 → 文件 → 新建场景 → 空场景
2. 保存为 `assets/scenes/Game.scene`（覆盖旧文件）

### 步骤 2: 创建 Canvas

1. 层级管理器 → 右键 → 创建节点 → Canvas
2. 确保 Canvas 的分辨率设置为 **400 x 700**

### 步骤 3: 创建 GameManager 节点

```
1. 在 Canvas 下创建空节点 → 命名为 "GameManager"
2. 添加组件 → TypeScript → GameManager
3. 添加组件 → FruitManager
4. 添加组件 → ScoreManager
5. 添加组件 → AudioManager
6. 添加组件 → DouyinSDK
```

### 步骤 4: 创建 FruitContainer 节点

```
1. 在 Canvas 下创建空节点 → 命名为 "FruitContainer"
2. 在 GameManager 组件中，将 FruitContainer 属性拖拽赋值
```

### 步骤 5: 创建 SpawnPoint 节点

```
1. 在 Canvas 下创建空节点 → 命名为 "SpawnPoint"
2. 位置设置为 (0, 600, 0)
3. 在 GameManager 组件中，将 SpawnPoint 属性拖拽赋值
```

### 步骤 6: 创建 UI 节点

```
1. 在 Canvas 下创建空节点 → 命名为 "UI"
2. 添加组件 → GameUI
3. 配置 GameUI 的各个属性（Label、Button 等）
```

### 步骤 7: 创建 Fruit 预制体

```
1. 在 assets/prefabs 文件夹右键 → 创建 → 预制体 → 命名为 "Fruit"
2. 打开预制体编辑
3. 添加节点结构：
   - Fruit (根节点)
     - Sprite (精灵节点)
     - CircleCollider2D (圆形碰撞体)
     - RigidBody2D (刚体)
4. 添加组件 → TypeScript → Fruit
5. 保存预制体
```

### 步骤 8: 配置 GameManager

```
在 GameManager 组件中：
- fruitPrefab: 拖入 Fruit 预制体
- spawnPoint: 拖入 SpawnPoint 节点
- fruitContainer: 拖入 FruitContainer 节点
```

### 步骤 9: 保存并测试

1. 保存场景 (Ctrl+S)
2. 点击运行按钮
3. 检查控制台日志

## ⚠️ 常见问题

### Q1: 找不到脚本类

**解决:** 菜单 → 开发者 → 刷新脚本

### Q2: 预制体无法创建

**解决:** 确保先有代码文件，再创建预制体

### Q3: 组件属性无法赋值

**解决:** 确保节点已添加到场景中
