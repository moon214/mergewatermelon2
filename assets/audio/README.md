# 音频资源目录

请在此目录放置音频文件：

## BGM (背景音乐)
- `bgm/game.ogg` - 游戏背景音乐

## SFX (音效)
- `sfx/drop.ogg` - 水果下落音效
- `sfx/merge.ogg` - 合成音效
- `sfx/combo.ogg` - 连击音效
- `sfx/fail.ogg` - 游戏结束音效
- `sfx/prop.ogg` - 道具使用音效

## 推荐格式
- 格式：OGG 或 MP3
- BGM: 128kbps, 立体声
- SFX: 64-128kbps, 单声道
