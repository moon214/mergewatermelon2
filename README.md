## 🚀 快速开始

### 1. 下载项目

```bash
git clone https://github.com/moon214/mergewatermelon2-douyin.git
cd mergewatermelon2-douyin
```

### 2. 打开抖音开发者工具

- 下载地址：https://developer.open-douyin.com/devtool
- 安装后打开

### 3. 导入项目

1. 点击「+」或「导入项目」
2. 选择项目目录：`mergewatermelon2-douyin`
3. 填入你的 AppID（如果没有，先注册抖音开放平台账号）
4. 点击「导入」

### 4. 预览/测试

- 点击工具栏「预览」按钮
- 扫码在抖音 App 中测试
- 或在模拟器中直接运行

## ⚙️ 配置说明

### 修改 AppID

编辑 `project.config.json`：

```json
{
  "appid": "YOUR_APPID_HERE"
}
```

替换为你的抖音小游戏 AppID。

### 配置广告位

编辑 `js/ad.js`，修改广告位 ID：

```javascript
this.adUnitIds = {
  rewarded: '你的激励视频广告位 ID',
  interstitial: '你的插屏广告位 ID',
  banner: '你的 Banner 广告位 ID',
};
```

**获取广告位 ID：**
1. 登录抖音开放平台：https://developer.open-douyin.com/
2. 进入「穿山甲」→「广告位管理」
3. 创建广告位，复制 ID

## 🎮 游戏玩法

1. **拖拽定位**：手指左右拖动调整水果位置
2. **点击下落**：松开手指，水果下落
3. **同等级合成**：两个相同水果碰撞后合成更高级水果
4. **避免溢出**：水果堆积超出红线则游戏结束

### 水果等级

## 📤 提交审核

### 准备工作

1. **游戏版号**：休闲小游戏可暂不需要
2. **隐私政策**：在抖音开放平台后台配置
3. **测试账号**：如有登录功能需提供

### 提交流程

1. 抖音开发者工具 → 点击「上传」
2. 填写版本号和备注
3. 登录抖音开放平台后台
4. 进入「版本管理」→ 提交审核
5. 等待审核（通常 1-3 个工作日）

### 审核注意事项

- ✅ 确保所有功能正常
- ✅ 广告位配置正确
- ✅ 无崩溃、无黑屏
- ✅ 符合抖音小游戏规范

## 🛠️ 开发调试

### 控制台日志

在抖音开发者工具中：
- 点击「调试」→「Console」
- 查看游戏日志

### 性能监控

- 帧率：保持 60fps
- 内存：< 100MB
- 启动时间：< 2 秒

### 常见问题

**Q: 图片加载失败？**
A: 检查 `images/` 目录是否有 8 个 PNG 文件

**Q: 广告不显示？**
A: 检查广告位 ID 是否正确，是否在抖音环境

**Q: 游戏无法启动？**
A: 检查 AppID 是否配置，是否导入成功
